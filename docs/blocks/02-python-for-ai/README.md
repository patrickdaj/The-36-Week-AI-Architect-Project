# 🐍 Unit 2 — Python for AI

<small>Weeks 3–4 · 1 block</small>

[→ Start Block 2](02-data-and-tooling/index.md)

---

You know Python. You've got a CS degree and you've automated things. But AI work has a specific flavor of Python — DataFrames, notebooks, vectorized operations, and a data-first workflow that's different from scripting or web dev.

This unit tunes your Python for the AI toolchain. By the end, you'll be fluent in Pandas, comfortable in Jupyter, and you'll have built a real security log analyzer that runs against actual data — no AI involved yet, just the data engineering foundation that makes everything else work.

**Why this matters:** Every later unit assumes you can load data, manipulate it, and explore it in a notebook. If you skip this, you'll waste time fighting Pandas when you should be learning RAG.

---

## What You'll Build

- An **automated security log analyzer** using Pandas on real public datasets
- **Data visualizations** of traffic patterns, alert frequencies, and anomalies
- A **CLI tool** that generates HTML reports from log data
- Your first **Jupyter notebook** workflow for data exploration
