# 🎯 Unit 1 — Prompt Engineering

<small>Weeks 1–2 · 1 block</small>

[→ Start Block 1–2](01-02-stop-wasting-tokens/index.md)

---

You use AI every day. You prompt Copilot, you chat with Claude, you paste things into ChatGPT. And most of the time you get *something* back — but you don't know why it worked or didn't, how much it cost, or how to make it repeatable.

This unit fixes that. You'll go from "spray tokens and hope" to writing prompts that are surgical, efficient, and reusable. You'll understand context windows, token economics, system prompts, and structured output — the mechanical skills that separate someone who *uses* AI from someone who *builds with* AI.

**Every later unit assumes you're prompting well.** This is the foundation.

---

## What You'll Build

- A **personal prompt library** organized by task type — debug, generate, review, explain, refactor
- **Security-specific system prompts** for config review, policy analysis, incident triage
- A **token cost calculator** script that tracks what your prompts actually cost
- A **prompt testing framework** — A/B test prompt variants and compare quality
